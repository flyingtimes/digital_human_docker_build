角色: 女白领
===================

这是 女白领 角色的文件夹。

需要添加的文件:
1. 参考音频文件 (支持的格式: .mp3, .wav, .m4a)
2. 参考图片文件 (支持的格式: .jpg, .png, .webp) 或 参考视频文件 (支持的格式: .mp4, .avi, .mov)

可选文件:
- config.json: 角色配置文件
- README.txt: 角色说明文件

配置说明:
- positive_prompt: 正面提示词
- negative_prompt: 负面提示词
- workflow_params: 工作流参数
- description: 角色描述
- tags: 角色标签

示例用法:
python character_manager.py generate 女白领 "要生成的文本内容"
